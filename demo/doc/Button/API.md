
## `props`
属性|类型|说明|默认值
---|---|---|---
`primary` | boolean | 主要按钮 | -
`text` | boolean | 文本按钮 | -
`cancel` | boolean | 取消按钮 | -
`disabled` | boolean | 禁止按钮 | -
`danger` | boolean | 危险按钮 | -
`className` | string | 类名 | -
`style` | object | 样式 | -
`onClick` | function | 点击时触发，参数为事件元素【设置了`disabled`时点击无效】 | -
