## `props`
属性|类型|说明|默认值
---|---|---|---
`className` | string | 类名 | -
`style` | object | 样式 | - 
`defaultValue` | string | 初始值 | -
`onChange` | function | 输入内容改变时触发，参数为输入内容 | -
`onBlur` | function | 失去焦点时触发，参数为输入内容 | -
`onFocus` | function | 获取焦点时触发，参数为输入内容 | -
`onPressEnter` | function | 按下回车时触发，参数为输入内容 | -
`placeholder` | string | 占位符提示 | `'请输入……'`
