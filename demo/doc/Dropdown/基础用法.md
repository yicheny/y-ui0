- `options` 下拉框选项
- `onChange` 选择项改变时触发的回调

### 示例

<!--start-code-->

```jsx
const options = Array.from(Array(100),(o,i)=>({value:i,text:`第${i+1}项`.repeat(Math.round(i/200 +1))}));

function Demo(){
    return <div>
            <Dropdown options={options} onChange={console.log}/>
    </div>
}

ReactDOM.render(<Demo/>);
```
