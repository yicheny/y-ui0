## `option`
属性|类型|说明|默认值
---|---|---|---
`options` | array | 下拉框选项，详细见下 | []
`onChange` | function | 选项改变时触发的回调，`(v,o)=>void` | -
`placeholder` | string | 占位提示 | `'请设置任意值...'`
`search` | boolean | 是否开启模糊搜索 | false
`defaultValue` | string / number | 初始值 | null
`disabled` | boolean | 是否开启禁用模式 | false
