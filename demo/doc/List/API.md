## `props`
属性|类型|说明|默认值
---|---|---|---
`className` | string | 类名 | -
`style` | object | 样式 | - 
`header` | string / element | 顶部内容 | - 
`footer` | string / element | 底部内容 | -
`data` | array | 列表项，详细见下 | -

## `data`
属性|类型|说明|默认值
---|---|---|---
`header` | string / element | 标题 | -
`bind` | string / element | 绑定属性 | -
