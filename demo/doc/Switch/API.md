## `props`
属性|类型|说明|默认值
---|---|---|---
`className` | string | 类名 | -
`style` | object | 样式 | - 
`defaultChecked` | boolean | 初始状态 | `false`
`onChange` | function | `boolean => void` | - 
`checkedChildren` | string / element |  选中时的内容 | `null`
`unCheckedChildren` | string / element | 非选中时的内容 | `null`
